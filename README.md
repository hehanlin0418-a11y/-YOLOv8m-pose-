camera_main.py：摄像头实时跌倒检测入口，调用系统摄像头并集成姿态检测、状态分析和可视化模块，支持异步曲线绘图与跳帧推理，适合直接运行进行实时监控演示。

collect_kpts_data.py：离线关键点序列采集工具，可从视频中逐帧提取纯YOLO、卡尔曼滤波、谱图插值或完整系统四种模式下的关键点坐标，并保存为 .npy 文件用于后续稳定性评估。

config.py：全局配置文件，定义了COCO关键点名称、人体骨骼连接关系、谱图拓扑邻接矩阵以及卡尔曼滤波和谱图重建的超参数常量。

eval_stability.py：稳定性评估与消融实验脚本，加载预先采集的关键点序列，计算帧间位移、有效点比例、躯干变异系数等指标，并生成双模型对比或四模型消融实验图表。

kalman_filter.py：分布式一致性卡尔曼滤波器实现，为每个关键点维护独立的状态估计，支持预测、测量更新以及与相邻滤波器的共识融合，用于提升关键点时序稳定性。

main.py：最简单的视频文件处理入口，调用 video_processor 对指定视频进行跌倒检测并输出带标注的结果视频。

multi_plotter.py：异步实时曲线绘图模块，在独立线程中动态更新躯干倾斜角、帧间位移、KF收敛度曲线以及脚踝运动轨迹，避免阻塞主检测循环。

pose_detector.py：姿态检测核心类，封装YOLOv8-Pose模型推理、分布式卡尔曼滤波平滑和谱图插值增强的完整流程，输出稳定可靠的17个关键点坐标与置信度。

pose_enhancer.py：谱图拓扑插值增强器，基于人体关键点邻接关系构建图拉普拉斯矩阵，对低置信度或缺失点进行邻域加权插值和对称补全，并计算帧间位移用于稳定性评估。

state_analyzer.py：跌倒状态分析器，通过人体宽高比、躯干倾斜角和垂直位置判断跌倒候选，结合滑动窗口与自适应阈值确认跌倒事件，并支持行走状态检测。

video_processor.py：完整的视频文件处理模块，整合姿态检测、状态分析、可视化绘制和实时曲线绘图，输出标注视频并记录模块协同日志。

visualizer.py：可视化绘制工具，在图像上绘制骨骼连线、颜色区分置信度的关键点以及当前状态、躯干角度、KF收敛度等文字信息。

Win.py：基于PyQt5的图形化集成界面，提供摄像头/视频文件选择、模型模式切换、进度拖拽、实时曲线显示和日志输出功能，是用户交互的主要入口。
